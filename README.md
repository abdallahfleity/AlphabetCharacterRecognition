# AlphabetCharacterRecognition
## 🧠 Dataset

This project uses the [EMNIST](https://www.nist.gov/itl/products-and-services/emnist-dataset) dataset.

To use it:
1. Download the dataset ZIP file from [https://www.nist.gov/itl/products-and-services/emnist-dataset)
2. Extract it into the folder:
3. Extract inside this folder 4 folders initial for the project that they are :
   emnist-mnist-test-images-idx3-ubyte
   emnist-mnist-test-labels-idx1
   emnist-mnist-test-train-idx3
   emnist-mnist-train-labels-idx1
